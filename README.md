# 🛍️ ShopSphere

ShopSphere is a full-stack MERN e-commerce application that enables users to browse products, manage their shopping cart, and securely authenticate, while providing administrators with a dedicated dashboard to manage products.

## 🚀 Live Demo

### User Website
🔗 https://shopsphere-frontend1.onrender.com/

### Admin Dashboard
🔗 https://shopsphere-admin4.onrender.com/addproduct

---

## ✨ Features

### 👤 User Features

- User Signup & Login
- Secure Authentication
- Browse Products
- View Product Details
- Add Products to Cart
- Remove Products from Cart
- Persistent Cart Storage
- Responsive User Interface

### 🛒 Shopping Cart

- Add products to cart
- Remove products from cart
- Cart data is stored in MongoDB Atlas
- Cart remains intact even after logout and subsequent login

### 🔐 Authentication

- User Registration
- User Login
- JWT-based Authentication
- Protected Routes

### 🛠️ Admin Features

- Dedicated Admin Dashboard
- Add New Products
- Upload Product Images
- Manage Product Inventory
- Delete Products
- Real-time Product Updates

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/ShopSphere.git
cd ShopSphere
```

### Backend Setup

```bash
cd Backend
npm install
npm start
```

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

### Admin Setup

```bash
cd Admin
npm install
npm run dev
```

## 🔄 Application Workflow

### User Workflow

1. Create an account or log in.
2. Browse available products.
3. View detailed product information.
4. Add products to the shopping cart.
5. Remove products from the shopping cart.
6. Cart data is stored in MongoDB Atlas.
7. Cart remains available even after logging out and logging back in.

### Admin Workflow

1. Access the Admin Dashboard.
2. Add new products to the platform.
3. Upload product images.
4. Manage available products.
5. Delete products when required.
6. Changes are reflected instantly on the user website.
